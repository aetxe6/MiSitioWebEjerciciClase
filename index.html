Detalles técnicos

  • Media query mínimo: @media (min-width:48rem) (768px).
  • Tipografía y medidas en rem para escalabilidad.
  • Imagen con srcset y sizes para ser responsiva.
  • Layout principal: CSS Grid (pasa a 2 columnas en escritorio).
  • Botones con borde visible y :focus claro para accesibilidad.
